**approve-bot** is a GitHub App which serves only to submit approving pull request reviews for release pull requests from [release-change](https://github.com/release-change/release-change).

## Installation

If you use release-change to manage the versions, the changelogs and the publishing of your packages and allow auto-merging of pull requests, you can [install a hosted version of this bot from the GitHub Apps marketplace](https://github.com/apps/release-change-approve).
